
#### Setup locally
```bash
# step 1
    - create open ai account
    - get api key
```

```bash
# step 2
    - create firebase account
    - setup firebase project
    - create firestore db
    - save config from project -> web section
```

```bash
# step 3
    - clone repo
    - do -> npm i 
    - change your configs of openai and firebase in 
        - environment
    - run locally -> npm start
```
